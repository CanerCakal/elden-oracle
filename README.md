# Elden Oracle

Elden Ring evreni hakkında soruları, yerel bir dil modeli ve
RAG (Retrieval-Augmented Generation) kullanarak yanıtlayan,
tamamen çevrimdışı çalışan bir soru-cevap asistanı.

## Teknolojiler
- Microsoft Foundry Local (cihaz üzerinde LLM)
- Python
- RAG mimarisi (embedding + vektör arama)
- SQLite (yerel veri saklama)
